# Perpetual Contract Project

Foundry-based skeleton for a perpetual futures contract.

## Structure

- `src/Perpetual.sol`: 主合约框架
- `test/Perpetual.t.sol`: 单元测试骨架
- `script/Deploy.s.sol`: 部署脚本示例

## Commands

- `forge build`
- `forge test`
- `forge script script/Deploy.s.sol --fork-url <RPC_URL> --broadcast`
